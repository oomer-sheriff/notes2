# Phase 1: Python Data Toolkit

> **Goal:** Fast-track refresher on NumPy and Pandas for ML workflows.
> **Duration:** 1 week

## Why You Need This
You know Python. But ML Python is different. It's all about vectorized operations. **Never use a `for` loop if NumPy can do it.**

## NumPy Essentials
- **Arrays (`np.array`)**: The fundamental data structure.
- **Broadcasting**: How NumPy handles operations on arrays of different shapes.
- **Vectorization**: Applying operations to whole arrays at once (fast, C-level).

## Pandas Essentials
- **DataFrames**: 2D labeled data structures.
- **Data Cleaning**: Handling `.isna()`, `.fillna()`, `.dropna()`.
- **Feature Engineering**: Creating new columns based on existing ones.

## Scikit-Learn Preprocessing
- `StandardScaler`: Scales features to mean=0, variance=1.
- `LabelEncoder` / `OneHotEncoder`: Categorical to numeric.
- `train_test_split`: Crucial for avoiding data leakage.
