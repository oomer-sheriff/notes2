# 🧠 Machine Learning Mastery — Learning Directory

> **Goal:** Go from Python backend dev to deep learning engineer who truly understands what's under the hood.  
> **Background:** Python (FastAPI), Docker, K8s, RAG/agents/chatbot experience — now learn the ML fundamentals powering it all.  
> **Framework:** PyTorch (primary) on Google Colab  
> **Outcome:** Able to build, train, fine-tune, and deploy ML/DL models from scratch.

## 🗓️ Quick Phase Overview

```
Phase 0  ── Math for ML (2-3 weeks)         → Linear algebra, calculus, probability
Phase 1  ── Data Toolkit Refresher (1 week)  → NumPy, Pandas, Matplotlib (quick)
Phase 2  ── Classical ML (3-4 weeks)         → Regression, trees, SVMs, clustering
Phase 3  ── Neural Networks (3 weeks)        → Perceptrons, backprop, PyTorch
Phase 4  ── CNNs & Vision (3 weeks)          → Convolutions, ResNet, YOLO
Phase 5  ── Sequences & RNNs (2 weeks)       → LSTMs, GRUs, seq2seq
Phase 6  ── Transformers (4 weeks) ⭐        → Attention, GPT, BERT, ViT
Phase 7  ── Generative Models (4 weeks) ⭐   → GANs, VAEs, Diffusion, DiT
Phase 8  ── NLP & LLM Theory (3 weeks)       → Theory behind RAG, fine-tuning, RLHF
Phase 9  ── MLOps (2 weeks)                  → Experiment tracking, model serving (light)
Phase 10 ── Advanced Topics (3 weeks)        → RL, GNNs, multimodal, MoE
Phase 11 ── Capstone Projects (4+ weeks)     → End-to-end builds for portfolio
```
